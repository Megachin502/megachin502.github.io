Flickr upload bot is a bot that allows any user to upload images to Commons. It will add to the page a template indicating who commanded the upload. Example. It will also be indicated below the information template:

The content of this image was reviewed by USER.

Please leave deletion notification on that user's talk page rather than the bot owner's. In case this is unclear or you have any questions, feel free to post on Bryan's talk page.
